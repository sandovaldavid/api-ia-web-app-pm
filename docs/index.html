API Intermediaria IA

Conectando Django y Ollama (DeepSeek) para optimizar la gestión de proyectos mediante IA

Ver API Docs Verificar Estado

Características Principales

🔄

Integración con Django

Consulta y actualiza datos de proyectos, tareas y recursos en Django de forma transparente.

🧠

Procesamiento con IA

Utiliza modelos de Ollama (DeepSeek) para análisis y recomendaciones inteligentes.

📊

Parametrización de Tareas

Analiza tareas para determinar su complejidad, tipo y tiempo estimado automáticamente.

👥

Asignación Inteligente

Recomienda los mejores recursos para cada tarea basándose en habilidades y disponibilidad.

💬

Sistema de Chat

Permite interactuar con la IA mediante conversaciones naturales y contextualizada.

🔒

Autenticación Segura

Sistema robusto de autenticación con JWT para proteger todos los endpoints.

Documentación de API

La API está completamente documentada utilizando OpenAPI/Swagger. Puedes explorar todos los endpoints, modelos de datos, parámetros y respuestas de manera interactiva.

Ver Documentación Swagger

Ejemplos de Uso

Parametrizar una tarea
GET /api/tasks/123/parameterize

Analiza una tarea y devuelve su complejidad, tipo, palabras clave y tiempo estimado.

Asignar recursos a un proyecto
GET /api/resources/assign/project/456

Recomienda el equipo óptimo para un proyecto completo y asigna recursos a cada tarea.

Flujo de Trabajo

  1. 🔐 Autenticación: El usuario se autentica para obtener un token JWT
  2. 📋 Proyectos: Se obtienen los proyectos y tareas desde Django
  3. 🧠 Análisis: La API procesa las tareas con IA (Ollama)
  4. 📊 Parametrización: Se analizan las tareas para determinar complejidad, tipo, etc.
  5. 👥 Asignación: Se recomiendan recursos para cada tarea
  6. 💬 Interacción: El usuario puede conversar con la IA sobre aspectos específicos
  7. 📝 Documentación: Se guarda el historial de todas las interacciones
